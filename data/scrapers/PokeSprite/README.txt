SCRAPES SMALL POKEMON SPRITES 2D (68x56)

download the folders:
https://github.com/msikma/pokesprite/tree/master/pokemon-gen7x
                              AND
https://github.com/msikma/pokesprite/tree/master/pokemon-gen8

using: https://download-directory.github.io/

and the json:
https://github.com/msikma/pokesprite/blob/master/data/pokemon.json

run python extractor.py to obtain all the sprite data locations
run python sorter.py to sort all those sprites into folders for each pokemon

NOTE:
pkmn.json and dex.json have useful translations for pokemon
